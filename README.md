### discourse-custom-emoji-priority

自定义表情符号优先级插件 for [Discourse](https://www.discourse.org). 将自定义表情符号显示在 emoji 选择器的顶部。

## 功能

- 将自定义表情符号（数据库中的 + 插件注册的）优先显示在每个分类的顶部
- 标准表情符号保持在原位
- 可通过站点设置开启/关闭

## 使用方法

在 emoji 选择器中，自定义表情符号将优先显示在每个分类的顶部。

## 安装

按照我们的 [Install a Plugin](https://meta.discourse.org/t/install-a-plugin/19157) 指南，使用以下命令：

```sh
git clone https://github.com/Jackzhang144/discourse-custom-emoji-priority.git
```

## 配置

在 `/admin/site_settings` 中找到 `custom_emoji_priority_enabled` 设置来开启或关闭插件。
